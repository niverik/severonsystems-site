# Severon Systems Static Site

Production-ready static site for GitHub Pages.

## Files
- `index.html`
- `style.css`
- `script.js`
- `.nojekyll`
- `assets/logo.png`
- `assets/erik-portrait.jpg`

## Deploy
1. Upload all files to the root of a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Set source to **Deploy from a branch**.
4. Choose `main` and `/ (root)`.
5. Save.

## Required edits before launch
- Replace the email address in `index.html` if needed.
- Replace the LinkedIn URL in `index.html` if needed.
- Confirm the portrait and logo are the intended final assets.
